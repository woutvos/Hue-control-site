Home

Hallo lezer! Momenteel werkt helaas nog niet alles, alles wat je ziet, op lege pagina's na, doet het wel!

Het kan zijn dat de pagina control page langer moet laden, gemiddeld duurt dit 5 seconden. Later word async toegevoegd om deze lang wachttijd op te lossen.

Vaak gestelde vragen

Ik heb geen hue bridge maar wil de control page wel bekijk, is dat mogelijk?

Jazeker! Ga naar de instellingen en selecteer bij "Change bridge" "example". Je kan nu de website bekijken zonder vervelende fouten te krijgen, je kan in deze modus helaas niks aanpassen.

De home pagina is in het Nederlands maar de rest in het Engels, hoe zit dat?

Momenteel word de site vooral in het engels gebouwd, behalve deze pagina. Later zullen we de mogelijkheid voor meerdere talen toevoegen.

Ik heb een idee voor de site, hoe kan ik dit doorgeven?

U kan het beste een issue aanmaken op Github.

Ik heb een probleem met de site, hoe kan ik dit doorgeven?

Ook daarvoor kunt u een issue aanmaken op Github.
